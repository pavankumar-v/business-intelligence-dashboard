6 KPIs

Highest Model Used - Most popular model by conversation count
Avg Spending Per Day - Average daily cost
Costliest Model - Model with highest total cost
Least Used Model - Model with lowest usage
Avg Token Consumption Per Day - Daily token usage
Model Efficiency - Completion/prompt token ratio (productivity)
Active Subscriber Utilization Rate - Percentage of active subscribers using the service

4 Charts

Spend Trend - Line chart (cost over time)
Cost Distribution By Model - Bar chart (cost per model)
Regional Cost Distribution - Pie chart (cost by region)
Company-wise Spend - Bar chart (top 10 companies by cost)
Active and Inactive Subscribers - Bar chart (active vs inactive)
